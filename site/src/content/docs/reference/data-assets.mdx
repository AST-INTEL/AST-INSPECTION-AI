## H30T Multi-Sensor Payload

The DJI H30T is a multi-sensor payload designed for professional inspection work. Each flight produces three co-registered image streams:

| Sensor | Specification | Inspection Use |
|---|---|---|
| RGB Wide | 48MP, wide field of view | Overall asset condition, spatial context |
| RGB Zoom | 36× optical zoom | Defect detail and confirmation |
| Thermal IR | Radiometric JPEG (RJPEG) | Hotspot detection, insulation failure, leakage |
| Laser Rangefinder | Distance measurement | Asset-to-drone distance metadata |

**Thermal note:** DJI RJPEG format embeds the raw temperature matrix in the JPEG file. Extracting it requires the DJI Thermal SDK. The visual JPEG alone is not sufficient for quantitative thermal analysis — the raw temperature data is needed.

## Data Security

Source data handling rules enforced throughout the system:

- Source archive is always mounted read-only — no AI process can modify original files
- Images and reports are never uploaded to any external service
- The vector database stores extracted text and embeddings, not original files
- All processing occurs on local hardware only
