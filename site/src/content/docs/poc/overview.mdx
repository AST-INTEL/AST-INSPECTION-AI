# Proof of Concept

## What the POC Is

The Proof of Concept is a fully functional end-to-end build of the AST Inspection AI system, running on a personal home workstation, using real CANAR inspection data.

It is not a prototype. It is not a mockup. It is not a demo environment with fake data. It is the complete system — vision pipeline, dual-LLM intelligence layer, vector database, Streamlit UI, security verification — running on slightly smaller models than production (14B instead of 32B) due to the 16GB VRAM constraint of the home machine GPU.

When the POC is done, a working Docker container exists that:
- Takes a folder of H30T images as input
- Runs them through the YOLOv11 vision pipeline
- Passes structured findings to the Inspection LLM
- Cross-references engineering standards from the vector database
- Produces a structured inspection report
- Can be copied onto a USB drive and deployed on a different machine in under an hour

## POC Hardware

| Component | Spec |
|---|---|
| CPU | AMD Ryzen 7 7800X3D |
| GPU | NVIDIA RTX 4080 Super — **16GB VRAM** |
| RAM | 64GB DDR5 |
| OS | Windows 11 + WSL2 (Ubuntu) |
| Runtime | Docker Desktop with GPU passthrough |

**VRAM constraint:** The 16GB VRAM means both LLMs cannot be loaded simultaneously. The Operations LLM (Qwen2.5-Coder 7B, ~4GB) loads first for system tasks, unloads, then the Inspection LLM (Qwen2.5 14B, ~9GB) loads for report generation. This adds latency but does not affect correctness. The production RTX 5090 (32GB VRAM) runs both simultaneously.

## POC Completion Criteria

The POC is complete when all of the following are true:

<Steps>
1. End-to-end pipeline processes a complete real inspection job without manual intervention
2. Generated report is reviewed by a qualified engineer and rated comparable in quality and completeness to a manually written report
3. Security verification test suite passes — Wireshark confirms zero outbound traffic during a 30-minute demo session
4. Cryptographic hash of the source archive before and after testing is identical — no file modification occurred
5. Docker container is exported, transferred to a second machine, loaded, and the smoke test suite passes
6. All phase documentation is complete and committed to the repository
</Steps>

<Aside type="tip">
The POC is not declared complete based on subjective confidence. It is declared complete based on these six checkboxes. All six must be ticked.
</Aside>
